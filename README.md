🎉 Aplicação de Confirmação de Presença - Meu Casamento 💍
Bem-vindo ao repositório da aplicação que desenvolvi para gerenciar as confirmações de presença do meu casamento! ❤️

Essa aplicação foi feita com muito carinho, unindo tecnologia e amor, para facilitar a organização desse momento tão especial.

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

React: Para a criação da interface interativa e amigável.
Node.js: Para o backend robusto e escalável.
Express: Framework minimalista para gerenciar rotas e APIs.
Axios: Para as requisições HTTP entre o frontend e o backend.
CORS: Para habilitar a comunicação segura entre o frontend e o backend.
Next UI: Para criar uma interface elegante e responsiva com componentes prontos.
